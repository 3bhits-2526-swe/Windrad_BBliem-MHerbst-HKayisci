### 🌬️ Windrad-System – Virtual Engineering (Bliem, Herbst, Kayisci)

**Das Konzept**
Das System visualisiert in Unity die Aerodynamik und den Wirkungsgradverlust (Wake-Effekt) von Windrädern, indem ein 3D-Objekt in ein dynamisches Windfeld platziert wird.

**Ablauf & Interaktion**
* **Input:** Windgeschwindigkeit (Slider), Windrad-Steuerung (An/Aus) und Objektauswahl (z. B. via QR-Code).
* **Logik:** Berechnung von Strömungsvektoren und Windschatten. Steht ein Rad still, bekommt das dahinterliegende mehr Wind.
* **Output:** Strömungspartikel, Wirkungsgrad-Anzeige und spürbarer Wind über einen Ventilator.
